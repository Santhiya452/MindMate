# 🧠 MindMate - AI Companion for Smarter Learning

MindMate is a full-stack AI-powered mental wellness and study companion app built for students.

## ✨ Features
- 🤖 AI Chat powered by Google Gemini
- 📧 Email OTP verification
- 😊 Mood tracker
- 💬 Chat history
- 🎤 Voice input
- ⚙️ Settings page
- 🏠 Beautiful Landing page

## 🛠️ Tech Stack
**Frontend:** React.js, Vite
**Backend:** Node.js, Express.js
**Database:** MongoDB Atlas
**AI:** Google Gemini API
**Auth:** JWT + Nodemailer OTP

## 🚀 How to Run
**Backend:**
cd backend
npm install
npm run dev

**Frontend:**
cd frontend
npm install
npm run dev
